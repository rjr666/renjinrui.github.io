Hello Blog

发表于 2018-11-25 | 分类于 我的博客之旅 |

前言

喜欢写Blog的人，会经历三个阶段。

  • 第一阶段，刚接触Blog，觉得很新鲜，试着选择一个免费空间来写。

  • 第二阶段，发现免费空间限制太多，就自己购买域名和空间，搭建独立博客。

  • 第三阶段，觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。

　　
————引自阮一峰

Blog

Github+Jekyll

 2018上半年的时候，学校在实训期，找的某个培训机构，给我们培训了前端的一些知识，然后自己就各种瞎折腾，偶然中发现了Github免费搭建博客的教程，然后照猫画虎利用github+jekyll copy了HyG的模板做了第一版。(目前已放弃)

Github+Hexo

 18年11月，重新开始做，利用github+hexo，hexo有自己的中文开发文档，hexo主页中有非常详细的介绍，我这里就不赘述了。我的个人博客目前使用的就是github+hexo。接下来我会介绍github+hexo搭建博客。

人生太精彩，我想用博客记录成长。

安装及其部署

Hexo 可以说是目前最流行的博客框架之一，基于Nodejs，更多信息可以google，下面需要安装的工具包括 Git，Nodejs，hexo。

  • Nodejs
  • Git

  • Hexo：以上所有都安装完成之后再使用git bash安装Hexo，当然你也可以使用Windows命令提示符完成安装。

  • 所有必须工具已经安装完成，下面我们就可以生成博客，并上传至我们的Github 仓库了。

接下来我们需要用Hexo初始化博客，然后更改一些自定义配置，或者加上自己喜欢的主题，写上第一篇文章，然后发布到自己的个人Github网站(username.github.io)

将下面的username可以替换为你自己的喜欢的名字（尽量使用英文），执行成功后，在目标的文件夹会出现一个username的文件夹
